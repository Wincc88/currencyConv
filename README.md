# Currency Converter
It will convert different currencies against 1 unit of the $ (United States dollars).
It will fetch information from some websites to get and update the OFFICIAL exchange rate daily (apex bank rate of country).
It will fetch information from some websites to get and update the parralel/black market or commercial bank forex rate daily
The daily rates will be put in tables while only the days rate shows. 

##
Different currencies can be checked with the drop-down feature
